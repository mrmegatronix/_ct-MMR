🔥 Cross-Device Remote (Firebase RTDB)

Enables the Remote Control to work across different devices on the same Wi-Fi network. Uses a free Firebase Realtime Database — no ongoing cost for this use case.

▶ How to set up Firebase RTDB (Free, ~2 minutes)
  1. Go to console.firebase.google.com and sign in
  2. Create or select your project → click Realtime Database → Create Database
  3. Choose a region, then start in test mode (or set rules to allow public read/write)
  4. Copy the Database URL — it looks like:
    https://your-project-default-rtdb.firebaseio.com
  5. Paste it above and click Save Firebase Config
  6. Reload the Display and Remote pages — they will now sync across devices
// Realtime Database Rules (Rules tab in Firebase console)
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
